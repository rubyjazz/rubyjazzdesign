Game Off Rebrand

The project was a yearly game development contest put on by GitHub and hosted on itch.io. The goal was to have aspiring developers create a game based on the theme Throwback in two weeks. The winner recieved recognition by GitHub as well as some juicy prize money.


The catch was that Game Off needed a complete rebrand as well as a yearly theme. For this, I delved into the essence of Game Off and created a logo that was cleaner, smarter, and friendlier than the privious iteration. The new logo uses git difference icons (the green plus and the red minus) to represent the buttons and d-pad on a controller. Those in turn create the 'O' in off.

After rebranding, the focus moved to making assets to promote that year's Game Off. The theme was Throwback, an homage to 80's gaming and culture. To make this feel both modern and retro, I used a neon color sceme, VHS tape distressing, and 80's objects. This game the gif and illustrations the funky nostalgic feeling I was looking for.
